# TuringCompleteBinaryRacer
Toy project for auto complete the binary racer minigame in Turing Complete

## Setup
1. Install [tesseract](https://github.com/tesseract-ocr/tesseract) and relevant dependencies required by the script.
2. Run
   ```bash
   python main.py
   ```
   from the repo directory.
3. Test if the script is working by pressing E on keyboard and observe the printout.

## Tested on
- OS: Windows
- Python version 3.10.19
- Screen resolution: 2440x1560
